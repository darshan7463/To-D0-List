# 📝 To-Do App

A simple and responsive To-Do List application built using HTML, CSS, and JavaScript. This project allows users to add tasks, delete tasks, and manage their daily to-do list with a clean and user-friendly interface.

## 🚀 Features

- Add new tasks
- Delete tasks
- Add tasks by pressing the **Enter** key
- Clean and responsive user interface
- Dynamic task creation using JavaScript
- Simple and easy-to-use design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

- Event Listeners
- Creating and removing HTML elements dynamically
- Working with Arrays
- JavaScript Functions
- Keyboard Events
- Building an interactive web application
